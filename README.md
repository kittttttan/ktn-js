## Working with [Grunt](http://gruntjs.com/)

install [node.js](http://nodejs.org/)

install Grunt

```bash
npm install -g grunt-cli
```

install project dependencies

```bash
cd /path/to/ktn-js
npm install
```

build

```bash
grunt
```

build examples for browser (with [browserify](http://browserify.org/))

```bash
grunt examples
```
